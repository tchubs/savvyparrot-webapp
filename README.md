# webapp

Web Application for SavvyParrot.com website.

## Getting Started

### Single-User Development

  ```sh
  $ git clone git@github.com:tchubs/savvyparrot-webapp.git
  ```

  ```sh
  $ npm install
  ```

  ```sh
  $ bower install
  ```

  ```sh
  $ lineman clean
  ```

  ```sh
  $ lineman build
  ```

  ```sh
  $ lineman run
  ```

  ```sh
  $ npm start
  ```
